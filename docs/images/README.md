## Redaction checklist (strict)
Before committing real captures, mask all sensitive data:
- Phone numbers
- Chat IDs / LID IDs
- API tokens and keys
- Private server URLs/IPs
- Bot tokens or session identifiers

## Style checklist
- Keep consistent width and readable crop
- Focus on relevant UI area only
- Ensure text is readable on GitHub
